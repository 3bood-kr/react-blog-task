## Getting Started

### 1. Clone the repository:
   ```bash
   git clone https://github.com/3bood-kr/react-blog-task.git
   ```

### 2. Change directory to the project folder:
   ```bash
   cd react-blog-task
   ```

### 3. Install dependencies:
   ```bash
   npm install
   ```

### 4. Start the development server:
   ```bash
   npm run dev
   ```
